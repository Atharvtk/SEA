# COVID-19 Data Analysis & Visualization

## Overview
This project provides a comprehensive analysis of COVID-19 trends across countries using visualizations. It highlights confirmed cases, deaths, recoveries, growth rates, and regional distributions, helping understand pandemic dynamics and identify hotspots.  

## Features
- Top 10 countries by confirmed cases, deaths, and recoveries  
- Weekly growth trends and new case analysis  
- Regional distribution by WHO regions  
- Deaths vs. confirmed cases scatter plots  
- Easy-to-read charts for data-driven insights  

## Technologies Used
- Python 3.x  
- Jupyter Notebook  
- Matplotlib / Seaborn / Pandas  
- IPython for displaying charts
